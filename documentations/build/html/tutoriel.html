1. Tutoriel rapide Utilisateur

Version : 1.2

1.1. Introduction

Visiopompe est un logiciel de suivi de pompe mécanique. Avant de débuter à enregistrer un stock de pompe, commencez par remplir dans l’ordre donné les différentes rubriques listées dans procédure.

1.2. Procédure pour créer un stock

Pour créer un stock, une équipe, un fabriquant, vous avez des formulaires intégrés dans des liens du menu de gauche. Dans le cas du panneau « d’administration », les formulaires sont intégrés aux pages directement.

En cliquant sur le formulaire, vous devez remplir chaque saisie. Les photographies ne sont pas obligatoires mais conseillées notamment pour les accessoires liés aux pompes comme les pièces détachées.

Note

Un stock d’une pompe a besoin pour être créé, à minima, d’un élément dans chaque rubrique, si cela n’est pas fait.

Voici chaque rubriques à remplir pour démarrer dans l’application si elle est vide sinon passez directement au point 2 ou 3 si le fabriquant de la pompe n’existe pas dans le formulaire de soumission.

  • 1.1 - Sites

  • 1.2 - Bâtiments

  • 1.3 - Etages

  • 1.4 - Pièces

  • 2 - [Menu] Administration -> Fabriquants

  • 3 - [Menu] Administration -> Modèles des pompes

  • 3.1 - Technologie de vide

  • 3.2 - Fiches de modèle (famille) de pompes

  • 4 - [Menu] Pompes -> Créer un stock

Avertissement

Il est rudement conseillé de compléter les autres élements pour un meilleur suivi de vos machines. (ex : les numéros d’inventaires..)

Veuillez vous référer à la documentation utilisateur complète pour plus de détails sur chaque panneau de gestion.

1.3. Consulter les stocks

Le menu de droite vous permet de consulter les différents via les liens présents et en cliquant sur les onglets « consulter ».

1.4. Un problème ?

Consulter la documentation utilisateur complète présente depuis le menu sinon contacter le support par mail.
